
# Job Application Website (HTML + CSS Only)

A clean, professional, and fully responsive **Job Application Website** built using **only HTML and CSS**.  
The site allows users to:

- View available job listings  
- Submit applications  
- Contact the company  
- Navigate easily with a modern interface  

This project focuses on **design, accessibility, and user-friendly layout** without using JavaScript.
